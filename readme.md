# Van Helsing Game

## JS13K@2022 - Death

### Controls:

- Forward: `W` or `ArrowUp`
- Back: `S` or `ArrowDown`

- Left (rotate): `A` or `ArrowLeft`
- Left (move): `Shift` + `A` or `ArrowLeft`
- Right (rotate): `D` or `ArrowRight`
- Right (move): `Shift` + `D` or `ArrowRight`

- Aim: `Mouse`
- Shot: `Click`

- Restart: `Esc`
